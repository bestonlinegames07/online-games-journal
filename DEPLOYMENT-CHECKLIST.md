# Pre-Launch Deployment Checklist

Use this checklist before deploying your white page to production.

- [ ] All blog posts are informational and educational
- [ ] NO gambling or real money gaming content
- [ ] NO misleading headlines or clickbait
- [ ] Content matches your ad vertical/topic
- [ ] Grammar and spelling checked
- [ ] Images load properly
- [ ] All text is readable (contrast, size)

## âœ… Technical Validation

- [ ] All internal links work (home, blog, about, etc.)
- [ ] No broken images or missing assets
- [ ] Cookie banner appears and can be dismissed
- [ ] Privacy policy and terms pages exist
- [ ] Sitemap.xml generated and valid
- [ ] Robots.txt allows crawling
- [ ] Mobile responsive (test on phone)
- [ ] Page loads in <3 seconds
- [ ] No console errors (F12 developer tools)

## âœ… Compliance Review

- [ ] Cookie consent banner functional
- [ ] Privacy policy present and complete
- [ ] Terms and conditions present
- [ ] No external links to non-compliant content
- [ ] Contact information (if required by jurisdiction)
- [ ] Age-appropriate content (family-friendly)
- [ ] GDPR considerations (if EU traffic)
- [ ] Disclaimer about informational nature

## âœ… SEO & Metadata

- [ ] All pages have unique titles
- [ ] All pages have meta descriptions
- [ ] Title tags are 50-60 characters
- [ ] Meta descriptions are 140-160 characters
- [ ] Images have alt text
- [ ] Proper heading hierarchy (H1, H2, H3)
- [ ] Sitemap includes all pages

- [ ] Chrome (desktop)

## âœ… Hosting & Domain

- [ ] Domain is registered and active
- [ ] SSL certificate installed (HTTPS)
- [ ] DNS properly configured
- [ ] No redirect loops
- [ ] WWW vs non-WWW decided and configured
- [ ] CDN configured (if using)
- [ ] Backup of site files stored

## âœ… Palladium Configuration

- [ ] Palladium account active
- [ ] Campaign created in Palladium
- [ ] White page URL entered correctly
- [ ] Money page URL entered correctly
- [ ] Filters configured (GEO, device, OS, etc.)
- [ ] Facebook IP blocks enabled
- [ ] Reviewer detection enabled
- [ ] Entry URL obtained from Palladium

## âœ… Testing Flow

### Test 1: Direct White Page Access
- [ ] Visit white page URL directly
- [ ] Verify all content loads
- [ ] Check mobile view
- [ ] Test all navigation links
- [ ] Dismiss cookie banner
- [ ] Visit privacy and terms pages

### Test 2: Palladium Entry URL (Residential IP)
- [ ] Use residential proxy or mobile data
- [ ] Visit Palladium entry URL
- [ ] Should redirect to MONEY page (if filters allow)
- [ ] Verify tracking/pixels fire

### Test 3: Palladium Entry URL (Datacenter IP)
- [ ] Use VPN or datacenter proxy
- [ ] Visit Palladium entry URL
- [ ] Should redirect to WHITE page
- [ ] Verify no money page elements visible

### Test 4: Facebook Ad Preview
- [ ] Create test ad in Facebook
- [ ] Use Palladium entry URL
- [ ] Preview ad
- [ ] Should show white page content
- [ ] No policy violations flagged
